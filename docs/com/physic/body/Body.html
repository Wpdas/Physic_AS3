Constructor Detail
Body()Constructor
public function Body(source:SpritePhysic, density:Number = 0)

Cria novo tipo de corpo. O objeto source deve ter originalmente o ponto de registro alocado ao Topo Esquerdo. O Pivot deve ser auterado (quando necessario) depois de instanciar o objeto através do metodo "updatePivot(Pivot.EIXO)" Para alterar os atributos de posição, rotação, escala e etc... do objeto. Use "objetoInstanciado.pivot.x" etc

Parameters
source:SpritePhysic — Objeto que vai ser inserido no corpo
 
density:Number (default = 0) — Densidade do objeto

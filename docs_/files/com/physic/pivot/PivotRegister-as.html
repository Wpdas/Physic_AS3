PivotRegister

ATENÇÃO: Esta classe é parte da lib Wenz.  Para mais informações acesse: http://migre.me/5NXoo Classe responsável por modificar o ponto de registro de um DisplayObject.  Todos os direitos estão reservados ao desenvolvedor.  Exemplo de uso: var objetoPoint:PointRegister = new PointRegister(objeto, Registers.CENTER); Existe a possibilidade de alterar o ponto de registro: objetoPoint.newRegister(Registers.MANUAL_POINT(10, 10));
